## DNS

DNS is managed through DreamHost.

Custom domain:

titus.elmfarm.co

DreamHost record:

Type: CNAME
Name / Host: @
Value / Points to: 664e5bd5bad399c4.vercel-dns-017.com

Note: In DreamHost, this record was added inside the DNS screen for `titus.elmfarm.co`, so `@` means `titus.elmfarm.co`.
